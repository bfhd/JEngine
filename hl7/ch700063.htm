7.18 RESTRICTIONS ON VALUATION OF OBX SEGMENT FIELDS

The result category for a given OBX segment determines how specific fields in that segment are valued. The following tables define field requirements for each category of OBX segment. The OPT/X column can take the values of R = Required, O = Optional, or X = Ignored and not valued. OBX Fields marked with an X should not be valued in Waveform response messages of specified Suffix type. Valuation of the fields must match the value provided in the associated wave category OBX segments, i.e., OBX with the same sub-ID must share the same result status.
