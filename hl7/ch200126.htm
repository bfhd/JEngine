2.24.3.0 ERR field definition

2.24.3.1 Error code and location (CM) 00024

Components: <segment ID (ST)> ^ <sequence (NM)> ^ <field position (NM)> ^ <code identifying error (CE)>

Definition: This field identifies an erroneous segment in another message. The second component is an index if there are more than one segment of type <segment ID>. For systems that do not use the HL7 Encoding Rules, the data item number may be used for the third component. The fourth component references a locally defined error table (as a CE data type) is restricted from having any subcomponents as the subcomponent separator is now the CE’s component separator.
