11.2.1 RQI/RPI - request for insurance information (event I01)

This event triggers a message to be sent from one healthcare provider to another to request insurance information for a specified patient. 

RQI     Request Patient Information    Chapter
  MSH    Message Header        2
 {
    PRD    Provider Data        11
    [{CTD}]   Contact Data         11
 }
  PID    Patient ID         3
[{NK1}]    Next of Kin         6
[[{GT1}]    Guarantor         6
  {
     IN1    Insurance         6
     [IN2]   Insurance, Additional info     6
     [IN3]   Insurance, Certification      6
  }
]
[{NTE}]    Notes and Comments       2
RPI     Return Patient Information    Chapter
  MSH    Message Header        2
  MSA      Message Acknowledgment      3
 {
    PRD    Provider Data        11
    [{CTD}]   Contact Data         11
 }
  PID    Patient ID         3
[{NK1}]    Next of Kin         6
[[{GT1}]    Guarantor         6
  {
     IN1    Insurance         6
     [IN2]   Insurance, Additional info     6
     [IN3]   Insurance, Certification      6
  }
]
[{NTE}]    Notes and Comments       2
