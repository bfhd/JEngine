The codes in HL7 table 0292 represent the initial content of the external code set CVX. Since immunizations may have to be added to this table more quickly than new versions of HL7 are released, this code system will be maintained by the Centers for Disease Control and Prevention. (Contact the Chief, Systems Development Branch, National Immunization Program, Centers for Disease Control and Prevention, 1600 Clifton Road, MS E-62,Atlanta, GA 30333; (404) 639-8245) http://222.cdc.gov/nip/home.html. When using this code system to identify vaccines, the coding system component of the CE field should be valued as "CVX", not as "HL70292."
