2.13.1 ACK - general acknowledgment

The simple general acknowledgment (ACK) can be used where the application does not define a special application level acknowledgment message or where there has been an error that precludes application processing. It is also used for accept level acknowlegements. The details are described in Section 2.12.1, "Original and enhanced processing rules."
