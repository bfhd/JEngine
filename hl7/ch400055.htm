4.12.1 VXQ -query for vaccination record (event V01)

Definition: When a state system does not already have the complete patient vaccination record, it will send a query (with a V01 event) for the definitive (last updated) record. Within the definitions for QRD and QRF, certain components are defined according to position in the field, as detailed in Section 4.11, "QUERIES FOR iMMUNIZATION RECORDS (QRF SEGMENTS)." The three-letter code in the leftmost column indicates the segment that is included; the column on the right specifies the chapter in which that segment is fully defined.

The query will follow this format: 

VXQ     Vaccination Query       Chapter
MSH     Message Header Segment      2
QRD     Query Definition Segment      2
[QRF]    Query Filter Segment       2
