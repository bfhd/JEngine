3.1 PURPOSE

The ADT transaction set provides for the transmission of new or updated demographic and visit information about patients. Since virtually any system attached to the network requires information about patients, the ADT transaction set is one of the most commonly used.

Generally, information is entered into an ADT system and passed to the nursing, ancillary and financial systems either in the form of an unsolicited update or a response to a record-oriented query.

This chapter defines the transactions that occur at the seventh level, that is, the abstract messages. The examples included in this chapter were constructed using the HL7 Encoding Rules.
