This section is current for Membership Ballot #2

For use notes, see Control section 2.4.3.7. When a table is used in an HL7 CE data type, its third component, "name of coding system" may have been adopted from some existing standard (for example, see list of tables in figures 2-2 and 2-3). If the table is not part of an existing standard, the third component, "name of coding system" constructed by appending the table number to the string "HL7". Thus, field RXR-administrative site, is a CE data type, which refers to table xxxx. Its "name of coding system" component is HL7xxxx.
