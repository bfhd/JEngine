11.2.3 RQI/RPR - request/receipt of patient selection list (event I03)

This trigger event occurs when the inquirer specifies a request for a listing of patient names. This event differs from event I02 in that it returns the patient list in repeating PID segments instead of repeating DSP segments. 

RQI    Request Patient Information     Chapter
  MSH    Message Header        2
 {
    PRD    Provider Data        11
    [{CTD}]   Contact Data         11
 }
  PID     Patient ID         3
[{NK1}]    Next of Kin         6
[[{GT1}]    Guarantor         6
  {
     IN1    Insurance         6
     [IN2]   Insurance, Additional info     6
     [IN3]   Insurance, Certification      6
  }
]
[{NTE}]    Notes and Comments       2
RPR     Return Patient List      Chapter
  MSH    Message Header        2
  MSA      Message Acknowledgment      3
 {
    PRD    Provider Data        11
    [{CTD}]   Contact Data         11
 }
[{PID}]     Patient ID         3
[{NTE}]    Notes and Comments       2
