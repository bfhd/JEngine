3.2.35 ADT/ACK - merge patient information - account number only (event A35)

Only the patient account number has changed as a result of the merge. See Section 3.5.2, "Merging patient/person information," for a discussion of issues related to the implementation of merge messages.

An A35 (merge patient information-account number only) event is intended for merging or changing an account number only.
